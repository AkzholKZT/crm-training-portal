# CRM Training Portal

Корпоративный веб-портал для обучения сотрудников отдела продаж.

## 🚀 Как запустить

### Вариант 1: Прямое открытие в браузере (рекомендуется)

Просто откройте файл `index.html` в любом современном браузере:
- Дважды кликните на `index.html`
- Или перетащите файл в окно браузера

### Вариант 2: Локальный сервер

Если вы хотите запустить локальный сервер:

**Windows PowerShell:**
```powershell
cd "C:\Users\Администратор\CascadeProjects\crm-training-portal"
python -m http.server 8000
```

Затем откройте в браузере: `http://localhost:8000`

## 📋 Функционал

### ✅ Реализованные разделы:

1. **Главная страница**
   - Приветствие
   - Быстрые ссылки на все разделы
   - Последние обновления
   - Новости и акции

2. **Обучение CRM**
   - Вход в CRM
   - Создание клиента
   - Создание заявки
   - Подключение домашнего интернета
   - Продажа SIM-карты
   - Добавление дополнительных услуг
   - Работа со статусами заявки

3. **Домашний интернет**
   - 4 тарифных плана с детальной информацией
   - Стоимость, скорость, описание
   - Оборудование и акции
   - Условия подключения

4. **Дополнительные услуги**
   - SIM-карты Activ (4 тарифа)
   - SIM-карты Altel (4 тарифа)
   - eSIM - инструкция по подключению
   - Семейные тарифы

5. **FAQ**
   - 6 вопросов по домашнему интернету
   - 10 вопросов по SIM-картам
   - Аккордеон-интерфейс для удобства

6. **База знаний**
   - Скрипты продаж
   - Инструкции
   - Документы
   - Ответы на возражения

7. **Новости**
   - 6 актуальных новостей
   - Категории: Тарифы, Акции, Важно, Технологии

8. **Контакты**
   - Форма обратной связи
   - Контакты администратора
   - Техническая поддержка

### 🔍 Дополнительные функции:

- **Глобальный поиск** по всем разделам
- **Адаптивный дизайн** для мобильных устройств
- **Современный интерфейс** с Bootstrap 5
- **Плавная навигация** между разделами
- **Интерактивные элементы** (аккордеоны, вкладки)

## 🛠️ Технологии

- **HTML5** - структура
- **CSS3** - стилизация
- **JavaScript** - интерактивность
- **Bootstrap 5** - UI фреймворк (через CDN)

## 💡 Особенности

- **Полностью локально** - работает без интернета
- **Без серверов** - статические файлы
- **Без платных API** - все данные в файлах
- **Без подписок** - разовое использование
- **Без внешних данных** - все данные включены

## 📝 Как редактировать контент

### Изменение тарифов
Редактируйте файл `data.js`, раздел `internetTariffs`

### Изменение FAQ
Редактируйте файл `data.js`, разделы `faqInternet` и `faqSim`

### Изменение новостей
Редактируйте файл `data.js`, раздел `news`

### Изменение стилей
Редактируйте файл `styles.css`
