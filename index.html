Обучение CRM

Пошаговые инструкции по работе с системой CRM

Инструкция по входу в систему

  1. Откройте браузер и перейдите по ссылке CRM-системы
  2. Введите ваш логин (обычно employee ID)
  3. Введите пароль
  4. Нажмите кнопку "Войти"
  5. При первом входе смените пароль на безопасный
Совет: Сохраните ссылку на CRM в закладках браузера для быстрого доступа

Как создать нового клиента

  1. В главном меню выберите "Клиенты" → "Добавить клиента"
  2. Заполните обязательные поля:
    • ФИО
    • Телефон
    • Адрес
    • Паспортные данные
  3. Выберите тип клиента (физическое/юридическое лицо)
  4. Нажмите "Сохранить"
Важно: Проверьте правильность введенных данных перед сохранением

Порядок создания заявки

  1. Выберите клиента из списка
  2. Нажмите "Создать заявку"
  3. Выберите тип услуги (интернет, SIM-карта, доп. услуги)
  4. Выберите тарифный план
  5. Укажите желаемую дату подключения
  6. Добавьте комментарии при необходимости
  7. Отправьте заявку на обработку

Процесс подключения интернета

  1. Создайте заявку на тип услуги "Домашний интернет"
  2. Выберите тариф из предложенных
  3. Укажите need оборудование (роутер)
  4. Проверьте техническую возможность подключения
  5. Согласуйте дату визита техника
  6. Информируйте клиента о сроках подключения

Продажа и активация SIM-карты

  1. Выберите оператора (Activ или Altel)
  2. Выберите тарифный план
  3. Сканируйте SIM-карту в системе
  4. Заполните данные клиента
  5. Проведите идентификацию клиента
  6. Активируйте SIM-карту
  7. Выдайте чек и инструкцию клиенту

Подключение дополнительных услуг

  1. Откройте карточку клиента
  2. Перейдите в раздел "Услуги"
  3. Нажмите "Добавить услугу"
  4. Выберите нужную услугу из списка
  5. Подтвердите добавление
  6. Информируйте клиента о изменениях

eSIM - виртуальная SIM-карта

eSIM - это встроенная SIM-карта, которая не требует физического носителя.

Преимущества:
  • Не нужна физическая SIM-карта
  • Можно использовать несколько номеров на одном устройстве
  • Быстрое подключение
  • Экологично
Как подключить:
  1. Проверьте совместимость устройства
  2. Обратитесь к менеджеру для получения QR-кода
  3. Сканируйте QR-код в настройках телефона
  4. Активируйте eSIM
Стоимость: Бесплатно при подключении тарифа

Семейные тарифы

Объедините всю семью на одном счете и экономьте!

Условия:
  • От 2 до 5 номеров на одном счете
  • Общие минуты и ГБ для всей семьи
  • Скидка до 20% на каждом номере
  • Единый счет для всех номеров
